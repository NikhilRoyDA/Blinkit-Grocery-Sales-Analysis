# Blinkit-Grocery-Sales-Data-Analysis

## Project Objective
The objective of this project is to analyze Blinkitâ€™s grocery sales data using Excel and provide valuable insights into sales performance, customer preferences, and outlet efficiency. The project addresses the following key business questions:
